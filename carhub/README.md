## Stack
- PHP 8.2+ with PDO (MySQL/MariaDB)
- MariaDB 10.4 via XAMPP
- Bootstrap 5.3.3, jQuery 3.7.1, Font Awesome 6.5
- Ajv 8 (client-side JSON Schema validation)
- opis/json-schema (server-side, optional — falls back without composer)

## Setup (XAMPP)

1. Drop project at `htdocs/carhub/` — folder name **must** be `carhub`.
2. Start Apache + MySQL in XAMPP.
3. Open phpMyAdmin → create database `carhub_db` → Import `carhub_db.sql`.
4. (Optional) Install opis/json-schema:
   ```
   cd htdocs/carhub
   composer install
   ```
5. Open `http://localhost/carhub/`.

## Assignment requirement coverage

| Requirement | Where |
|---|---|
| HTML/CSS + Bootstrap responsive | `css/style.css`, all `frontend/pages/` |
| Client-side validation (HTML5 + JS) | `assets/js/validation.js` |
| PHP server-side validation | `backend/validation/` |
| Secure file uploads | finfo MIME check, bin2hex rename, PHP blocked in uploads/ |
| CRUD + database | PDO prepared statements, transaction on listing create |
| Admin panel + RBAC | `admin_dashboard.php` + `require_role()` everywhere |
| Full-stack security | CSRF, bcrypt, session regen, 30-min timeout |
| **jQuery + AJAX** | Home carousel, browse filter/pagination, dashboard actions |
| **JSON file produced** | `export_listings.php` → `data/listings.json` |
| **JSON consumed by PHP** | `consume_listings.php` |
| **JSON consumed by jQuery/AJAX** | Admin dashboard `$.getJSON` + Ajv |
| **JSON Schema at creation** | `export_listings.php` validates before writing |
| **JSON Schema at consumption** | PHP (`consume_listings.php`) + Ajv 8 (admin dashboard) |
| **Laravel** | Separate `carhub-laravel/` repo |

## Demo flow for the video

1. **Frontend** — open home page, show hero + AJAX carousel with dot navigation.
2. **Browse + AJAX** — live filter + pagination; open Network tab to show JSON response.
3. **Client-side validation** — submit empty/bad form fields, JS blocks with alerts.
4. **Server-side + CRUD** — register Seller, log in, create listing (drag-and-drop images, set cover). Check `uploads/cars/` for bin2hex filenames.
5. **Admin panel** — log in as admin, approve listing, show `require_role` blocking non-admins.
6. **Messaging** — contact seller as Buyer; switch to seller, show Inbox with unread badge.
7. **JSON + Schema** — Admin dashboard: click Export (validates at creation, shows result). Click Consume + Validate (jQuery fetches file, Ajv validates client-side).
8. **AJAX events** — seller dashboard: Mark Sold (row updates), Delete (row fades out).
9. Cut to **Laravel demo**.
